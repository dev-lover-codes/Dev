HTML Lists Tutorial

Lists in HTML are used to display a collection of items, such as tasks, steps, or related data.

1. Unordered List (`<ul>`)

Unordered lists are used when the order of the items does not matter. Each item is marked with a bullet point.

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

Rendered Output:

  • Apples
  • Bananas
  • Cherries

2. Ordered List (`<ol>`)

Ordered lists are used when the order of the items does matter. Each item is numbered automatically.

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Rendered Output:

  1. First Item
  2. Second Item
  3. Third Item

Key Notes:

  • Use <ul> for bullet-pointed lists.
  • Use <ol> for numbered lists.
  • Each item inside a list must be wrapped in a <li> tag.
  • You can also nest lists inside each other.
