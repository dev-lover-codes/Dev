HTML Tables with thead/tbody and CSS Properties Explained

This webpage demonstrates HTML tables with <thead> and <tbody>, along with CSS styling, with explanations embedded in the page.

1. CSS for Page Styling

Explanation: The <style> tag in the <head> defines CSS rules to control the page's appearance, including fonts, colors, and layout. 
body {
  font-family: 'Times New Roman', Times, serif, sans-serif; /* Sets font for the page */
  padding: 20px; /* Adds space inside the page */
  background-color: #f4f4f4; /* Light gray background */
  color: #333; /* Dark gray text color */
}

2. Table Showing CSS Properties

Explanation: This table uses <thead> to group header rows and <tbody> for data rows, improving structure and accessibility.
CSS Property What It Controls Example Value Purpose
font-family Text font style 'Times New Roman', Times, serif Improves readability & appearance
padding Space inside element 20px Creates breathing room inside the box
background-color Background shade #f4f4f4 Softens the background
color Text color #333 Makes text readable
Explanation: <thead> wraps the header row(s) for better semantics and styling control. <tbody> groups data rows, making the table structure clearer for browsers and screen readers.

3. The <pre> Tag

pre {
  background-color: #e9e9e9; /* Light background for code */
  padding: 10px; /* Inner space */
  border-left: 4px solid #0066cc; /* Blue left border */
  overflow-x: auto; /* Scroll if code is too wide */
}
Explanation: The <pre> tag preserves text formatting (spaces, line breaks), ideal for displaying code snippets with proper indentation.

4. HTML Table Example with thead/tbody

Explanation: This table uses <thead> for headers and <tbody> for data, showing a structured table with sample data. 
<table>
  <thead>
    <tr><th>Name</th><th>Age</th><th>City</th></tr>
  </thead>
  <tbody>
    <tr><td>Alice</td><td>25</td><td>New York</td></tr>
    <tr><td>Bob</td><td>30</td><td>Los Angeles</td></tr>
  </tbody>
</table>

Rendered Output:

Name Age City
Alice 25 New York
Bob 30 Los Angeles
Explanation: <thead> groups the header row, making it distinct for styling or scripting. <tbody> organizes data rows for clarity and accessibility.

5. Explanation of Table Tags

Explanation: This table lists HTML table tags, including the new <thead> and <tbody>, with their purposes.
Tag Meaning
<table> Creates the entire table container
<thead> Groups header rows for better structure and styling
<tbody> Groups data rows for clarity and accessibility
<tr> Adds a new row
<th> Header cell (bold, centered)
<td> Regular data cell
Note: <thead> and <tbody> enhance table semantics, making it easier to style headers differently or manipulate data via JavaScript.

6. Nested Table (Table inside Table)

Explanation: This example shows a nested table with <thead> and <tbody> inside an outer table cell. 
<table>
  <thead>
    <tr><th>Outer Column</th></tr>
  </thead>
  <tbody>
    <tr>
      <td>
        Outer Row
        <table>
          <thead>
            <tr><th>Nested Header</th></tr>
          </thead>
          <tbody>
            <tr><td>Nested Table Cell</td></tr>
          </tbody>
        </table>
      </td>
    </tr>
  </tbody>
</table>

Rendered Output:

Outer Column
Outer Row
Nested Header
Nested Table Cell
Explanation: A nested table is placed inside a <td>. Using <thead> and <tbody> in both tables improves structure and readability.
